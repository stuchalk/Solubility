JSmol is an HTML5 version of Jmol. This page was written to accompany the article "Online tools for crystallographic symmetry handling and visualization". It demonstrates all of the Jmol scripting commands described in that paper. Drag the molecule around using the left mouse button. Scroll and click on various links on the right, below. They are designed to be clicked sequentially from top to bottom, starting at any link that is in bold. But you can try them in any order! Experiment!
